# AI Agent Admin 🐈

A local management dashboard for the configuration, memory, and personality of your **Nanobot** AI agent.

## Features

- **Providers Management**: Easily manage your AI model providers, API keys, and custom API bases.
- **Configuration Management**: Direct editing of `config.json` and core system files.
- **Skills Center**: List, create, and manage specialized agent skills with a simplified creation workflow.
- **Workspace Memory**: Visualization and management of the agent's knowledge base (`AGENTS.md`, `MEMORY.md`, `SOUL.md`, etc.).
- **Core Files**: Quick access to the agent's "soul" and identity files.
- **Tasks & Routine**: Direct access to `HEARTBEAT.md` for managing automated agent tasks.

## Tech Stack

- **Next.js 15** (App Router)
- **Tailwind CSS 4**
- **Lucide React** (Icons)
- **Shadcn UI** (Components)

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to access the dashboard.
